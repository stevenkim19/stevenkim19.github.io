Collaborative Graphical Editor


Developed in Java, the Collaborative Graphical Editor to allow multiple users to design and edit simple designs at the same time.

There are five components to this project:

  • An editor sends requests to, according to the user's commands, and receives requests from the editor communicator. Its state is updated according to the command the editor communicator passes on.
  • An editor communicator sends requests to and receives requests from the sketch server communicator.
  • Handle message is a middleman that intercepts requests to the editor communicator and the sketch server communicator. Because all forms of communication is done through strings, it analyzes which commands need to be executed according to the string that is passed.
  • A sketch server communicator sends requests to the server and receives requests from the editor communicator.
  • The sketch server updates the overall state and passes it onto clients.

Four options were implemented: 'draw', 'move', 'recolor', and 'delete'. The user can draw an ellipse, rectangle, a polyline(free drawing line), or a straight line. The user can also select any color of his/her choice.
