Speech Tagger



The goal of this project was to build a simple personal digital assistant that can identify a given word’s part of speech (POS) in a given context.

Just as a machine learns through training, the assistant learns through test files. There are two per test: a regular text file with English sentences and another that has the POS match of the former. For instance, if the test file has the sentence “I am smart”, then the latter file will have “N V ADJ”, with each POS separated by a space.

The way that the files are tested is simple. A Hidden Markov Model (HMM), an algorithm for determining the best path, is generated based on the frequencies of how many POS follows another. For example, from the database of tests, if there are 2 cases in which an adverb follows a verb, 5 in which a noun follows, and 0 cases in which a verb follows, then there is a probability of 5/(2+5+0) = 5/7 that the current word is a noun. Comparing the probabilities, the assistant identifies the word accordingly. Therefore, an increase in the number of sentences tested will yield a better prediction, as the database from which the assistant can search through increases.
